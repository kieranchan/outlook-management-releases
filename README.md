## English

This repository is a **release-only distribution repository** for `Outlook Management`.
It is used to share packaged binaries only and does **not** include source code.

### Download
Please go to the **Releases** page of this repository to download the latest packaged binaries.

### Available editions
- Modern edition (recommended)
- Classic edition (fallback)
- Release notes
- SHA256 checksum file

### Version notes
- Modern edition: recommended by default, uses the newer three-panel layout and supports multiple themes.
- Classic edition: kept as a compatibility / fallback version.

### How to use
1. Download the zip package you need from the **Releases** page.
2. Extract the entire archive to a local folder.
3. Open the extracted folder.
4. Run the `.exe` file inside it.

### Notes
- Keep the full extracted folder structure. Do not run the `.exe` alone after separating it from its folder.
- For normal use, the modern edition is recommended.
- If you encounter compatibility issues with the modern edition, try the classic edition.

### Verification
You can verify file integrity with `SHA256SUMS.txt`.
