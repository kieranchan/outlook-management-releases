# Release Notes / 发布说明

## v1.0.0 · 2026-03-06

### 中文

#### 本次发布
- 发布新版与经典版两个可执行成品包
- 发布独立的 GitHub 成品仓库，便于直接下载和分享
- 提供 SHA256 校验文件，便于校验完整性

#### 新版
- 使用 Qt 左中右主布局
- 支持多主题切换
- 支持窗口状态记忆
- 支持批量刷新 Token
- 支持批量并发拉取邮件
- 支持失败账号重试
- 修复下拉框选项显示问题
- 修复并发拉取邮件不可用的问题

#### 经典版
- 保留为稳定备用界面
- 支持独立启动
- 可用于兼容或回退场景

#### 发布附件
- `outlook-management-modern-win64-20260306.zip`
- `outlook-management-classic-win64-20260306.zip`

### English

#### Release Summary
- Ships both the modern and classic executable packages
- Publishes a standalone GitHub release repository for direct download and sharing
- Includes a SHA256 checksum file for integrity verification

#### Modern edition
- Uses the Qt three-panel layout
- Supports multiple themes
- Persists window state and UI preferences
- Supports batch token refresh
- Supports concurrent batch mail fetching
- Supports retrying failed accounts
- Fixes the dropdown visibility issue
- Fixes the concurrent mail fetch issue

#### Classic edition
- Kept as a stable fallback UI
- Can be launched independently
- Suitable for compatibility or rollback scenarios

#### Release assets
- `outlook-management-modern-win64-20260306.zip`
- `outlook-management-classic-win64-20260306.zip`
